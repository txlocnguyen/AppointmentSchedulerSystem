	WGU C195 - SOFTWARE II - ADVANCED JAVA CONCEPT

- This is a Java application built with JavaFX, and it is connected to a local MySQL database. It acts as a GUI-based appointment scheduling desktop appplication. 
- Author: Loc Nguyen
- Contact information: lngu242@wgu.edu
- Application version: 2.0.0
- Date: August 31st, 2022
- IDE used: IntelliJ IDEA Community Edition 2021.1.3 x64
- JDK version 17.0.1
- JavaFX SDK version 17.0.1
- MySQL Connector Java version 8.0.25
- The program should be able to run fine out of the box. The application entry point is Main.java. Please make sure JavaFX library and MySQL Connector Java Driver is properly installed under Project Structure.
- Additional report chosen: A report of appointments by Customer. It's very useful if we want to look up the list of every appointment that are associated with a specific customer.
- Login tracking file is saved to login_activity.txt which is located in the root folder of the project
- JavaDoc is located in src/main/JavaDoc
- Lambda expressions used in this program are located under:
        1/ initialize method in src/main/java/com/locnguyen/appointmentschedulerlocnguyen/controller/appointmentAddingController.java and src/main/java/com/locnguyen/appointmentschedulerlocnguyen/controller/appointmentModdingController.java
        2/ deleteApptClicked method in src/main/java/com/locnguyen/appointmentschedulerlocnguyen/controller/mainMenuController.java
